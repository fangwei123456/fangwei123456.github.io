# Wei Fang's Homepage

Personal academic homepage powered by [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme.

## Local Development

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve
```

The site is configured for GitHub Pages at <https://fangwei123456.github.io/>.
