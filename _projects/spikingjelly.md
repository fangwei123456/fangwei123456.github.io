[SpikingJelly](https://github.com/fangwei123456/spikingjelly) is an open-source deep learning framework for Spiking Neural Networks. I am one of the main contributors.

The project provides infrastructure for spike-based intelligence research, including neuron models, network training utilities, ANN-to-SNN conversion, event-based processing, and efficient simulation workflows.
